## Simple-Calclator

A tool for performing four arithmetic operations on a console created in C++.  
This is one of the projects that also serves as C++ practice!

## Compile

Compilation with [MinGW](https://sourceforge.net/projects/mingw/)

```
g++ calc source.cpp
```
